核心实现逻辑

1. 界面初始化

创建主窗口和各类控件（按钮、输入框、文本显示区），设置样式和布局，初始化信号槽连接。

2. 模型加载

加载预训练的Qwen2.5-VL模型，初始化语音识别模型，设置文本转语音引擎。

3. 对话流程

用户输入（文本/语音/图像） → 预处理输入数据 → 调用模型生成响应 → 显示结果并可选语音播报。

4. 多线程处理

使用多线程处理音频录制、语音识别和模型推理，确保UI线程不被阻塞。

皮肤美白处理

基于深度学习的智能皮肤美白系统

结合人脸检测、皮肤区域分割和图像处理技术，实现自然美白效果

项目概述

本程序是一个基于深度学习的智能皮肤美白处理系统，结合了先进的人脸检测、皮肤区域分割和图像处理技术，能够自动识别图像中的人脸皮肤区域并进行自然的美白处理。

注意：美白处理保持了皮肤的质感和细节，避免了过度处理导致的不自然效果。

核心功能

人脸检测

使用YOLO模型精准定位图像中的人脸位置

皮肤分割

基于U-Net架构实现高精度的皮肤区域分割

智能美白

应用先进的图像处理算法实现自然美白效果

参数可调

提供多个调节参数，满足不同美白需求

技术架构

U-Net皮肤分割

采用经典的U-Net架构，包含编码器（特征提取）和解码器（特征重建）部分，通过跳跃连接保留空间信息，实现精确的皮肤区域分割。

美白算法

基于多项式变换的美白公式：

output = (-2 * whiten_mask + 1) * img² + 2 * whiten_mask * img

通过调整掩码权重控制美白强度，结合形态学操作确保过渡自然。

使用说明

环境要求

  • Python 3.x
  • PyTorch
  • OpenCV
  • Ultralytics YOLO
  • NumPy

参数调整

  • expand_pixels: 控制人脸检测框的扩展范围
  • whitening_degree: 调整美白强度（0.0-1.0）
  • kernel_size: 控制掩码平滑程度

应用场景

  • 人像摄影后期处理
  • 美容应用开发
  • 图像编辑工具
  • 人脸分析系统
